Названия страниц с фильмами определенного года/жанра/страны/актера/режиссера/типа/поискового запроса

Написание названий для этих страниц имеют практически одинаковую структуру, единственное отличие, это наличие разных кодов. Коды нужны для того, чтобы система зная на какой странице находится, выводить соответственное название. К примеру код [year] будет отображаться в категории фильмов 2017 года, как 2017, а код [genre] в категории комедийных фильмов, будет комедия. Таким образом, написав название Лучшие фильмы из страны [country] онлайн, Вы сделаете название страницы с фильмами из США - Лучшие фильмы из страны США онлайн. Если Вам нужно написать отдельное название для какой-то категории, например для типа фильмы, Вы хотите отдельное описание, то нужно написать так:
(фильмы) {Лучшие фильмы онлайн бесплатно в хорошем HD качестве}
(default) {[type] онлайн бесплатно}
Таким образом на странице с типом фильмы название будет Лучшие фильмы онлайн бесплатно в хорошем HD качестве, а на страницах всех остальных типов, названия будут браться из default, например на странице сериалы, название будет сериалы онлайн бесплатно. По такому же принципу можно писать описания для отдельных жанров, стран и т.д. Просто указывайте в скобках, для какой категории Вы пишите описание (комедия), (2017), (Том Круз), а в фигурных скобках описание для этой категории. Не забывайте о коде (default), чтобы из него брались описания всех остальных категорий.

Названия страниц со списком всех годов/жанров/стран/актеров/режиссеров

Напишите названия для страниц со списком всех категорий, будь то список всех жанров, стран, актеров или режиссеров.

Название на странице фильма

Данное название будет отображаться на страницах всех фильмов на сайте. Благодаря кодам, Вы можете создать уникальное название, к примеру, чтобы создать название такого вида Онлайн фильм Матрица с участием Киану Ривз Вы должны написать Онлайн фильм [title_ru] с участием [actor]. Учтите, что в коде [actor] будет выбран случайный актер и списка главных актеров и не всегда это будет Киану Ривз.

Важно заметить

База фильмов CinemaPress - это исключительно свободно доступная информация о всех фильмах в мире, которую можно получить с TheMovieDB и подобных ему API-сервисов. Физически фильмы хранятся на открытых и закрытых видео-хостингах и владелец сайта сам решает, какие использовать, а какие отключить. Мы не поощряем пользователей к игнорированию владельцев прав или РКН и всячески рекомендуем им включить автоматическое удаление системой защищенных фильмов.
